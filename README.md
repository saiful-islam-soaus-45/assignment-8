# Online Book Borrowing platform


---

##  Project Overview
This is a modern **Online Book Borrowing platform web application** built for  Student  
It allows users to browse books, view details, and manage interactions through an intuitive and responsive UI.

The project focuses on clean UI design, fast performance, and smooth user experience using modern frontend tools.

---

##  Key Features
-  Browse books with category filtering
-  Detailed book view page
-  Borrowing cart system
-  Authentication system (BetterAuth)
-  Fully responsive design (mobile + desktop)
-  Fast UI with Next.js App Router
-  Modern UI using Tailwind + DaisyUI/HeroUI components

---

##  Tech Stack
- **Next.js** – React framework for SSR & routing
- **Tailwind CSS** – Utility-first styling
- **DaisyUI / HeroUI** – Prebuilt UI components
- **BetterAuth** – Authentication system
- **React Icons / Lucide React** – Icon library
